  • Maintenance Config: Gestione maintenance configurations
  • Repository: Solution - Azure Update Manager

Azure Posture Assessment (WAF/CAF)

Assessment read-only della subscription allineato al Well-Architected Framework: analizza Azure Advisor (5 pilastri), Secure Score, compliance delle Azure Policy e governance risorse, con readiness score e action plan. Richiede solo il ruolo Reader.

  • Well-Architected Framework: Microsoft Learn — Azure Well-Architected
  • Azure Advisor: Microsoft Learn — Azure Advisor
  • Cloud Adoption Framework: Microsoft Learn — CAF
  • Endpoint: /api/assess-azure-posture — read-only, nessuna modifica alle risorse.

Azure Cost Optimization

Assessment read-only focalizzato su sprechi e risparmio: raccomandazioni Cost di Azure Advisor con stima savings annui, dischi gestiti orfani, IP pubblici non associati, snapshot obsoleti e resource group vuoti. Richiede solo il ruolo Reader.

  • Cost optimization: Advisor — Cost recommendations
  • Cost Management: Microsoft Learn — Cost Management
  • Endpoint: /api/assess-azure-cost — read-only, nessuna risorsa eliminata.

Entra ID Identity Assessment

Assessment tenant-wide read-only della postura identity di Entra ID via Microsoft Graph: ruoli privilegiati (Global Admin sprawl), Security Defaults vs Conditional Access, copertura MFA, blocco legacy auth, igiene app registration e governance guest.

  • Identity secure score: Microsoft Learn — Identity Secure Score
  • Conditional Access: Microsoft Learn — Conditional Access
  • PIM (privileged roles): Microsoft Learn — Privileged Identity Management
  • Scope Graph richiesti: Directory.Read.All, Policy.Read.All, RoleManagement.Read.Directory, Application.Read.All, AuditLog.Read.All.
  • Endpoint: /api/assess-entra-identity — read-only, nessuna modifica al tenant.
