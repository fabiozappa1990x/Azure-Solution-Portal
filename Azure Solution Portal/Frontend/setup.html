Azure Function App (Backend Precheck)

Function App esistente nella subscription
Verifica tramite Azure Management API che la Function App sia presente
Function App non trovata — esegui Bootstrap.ps1:

1. Apri PowerShell 7 sul tuo PC
2. Naviga nella cartella del progetto
3. Esegui: .\Bootstrap.ps1
4. Lo script crea automaticamente Function App, App Registration e configura tutto
5. Torna qui e clicca Ricontrolla
Function App raggiungibile (CORS)
La Function App deve accettare richieste cross-origin dal dominio corrente
Come configurare CORS:
Opzione A — Azure Portal: Function App → Settings → CORS → Aggiungi
Opzione B — Azure CLI: az functionapp cors add --name NOME --resource-group RG --allowed-origins *
Tutti gli endpoint precheck attivi
Le 5 funzioni (Monitor, AVD, Backup, Defender, Updates) devono essere deployate e rispondere

Azure OpenAI (Analisi AI)

Azure OpenAI configurato (analisi AI)
L'istanza Azure OpenAI deve essere attiva e le credenziali configurate nei precheck scripts
Azure OpenAI non configurato:
Bootstrap.ps1 crea automaticamente l'istanza OpenAI e aggiorna le credenziali.
Se hai già un'istanza, aggiorna manualmente $endpoint e $apiKey nei file FunctionApp/scripts/*.ps1 e fai un nuovo push.
