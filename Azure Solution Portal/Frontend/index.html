Azure Monitor

Monitoraggio completo per VM Azure: dashboard KQL, alert CPU/Memoria/Disco, Data Collection Rules e policy di auto-enrollment.

  • Dashboard con 4 tab (Panoramica, Performance, Alert, Disponibilità)
  • Alert CPU, Memoria, Disco, Heartbeat
  • Azure Monitor Agent + DCR automatici
  • Policy auto-enrollment nuove VM

Azure Virtual Desktop

Desktop virtuale enterprise con Host Pool, Session Hosts, FSLogix per profili utente, Scaling Plan automatico e integrazione Entra ID.

  • Host Pool Pooled / Personal
  • Session Hosts Windows 11 multi-session
  • FSLogix su Azure Files (AADKERB)
  • Scaling Plan automatico
Disponibile

Azure Backup

Protezione dati enterprise con Recovery Services Vault, policy GFS (Daily/Weekly/Monthly/Yearly), soft delete e auto-protezione tramite Azure Policy.

  • Recovery Services Vault (GRS/LRS)
  • Policy GFS configurabili
  • Enhanced Policy (backup orario)
  • Auto-protezione VM con tag
Disponibile

Microsoft Defender for Cloud

Postura di sicurezza cloud con piani Defender attivabili (Server, Storage, Key Vault, ARM), contatti di sicurezza e auto-provisioning MDE.

  • Defender per Server P1/P2 (con MDE)
  • Defender CSPM
  • Auto-provisioning MDE + AMA
  • Microsoft Cloud Security Benchmark
Disponibile

Microsoft Intune

Analisi completa del tenant Intune: inventario dispositivi gestiti, app rilevate su ogni device e applicazioni deployate con stato di assegnazione.

  • Inventario dispositivi Windows, iOS, Android, macOS
  • App rilevate con versione e diffusione
  • App deployate e stato assegnazione
  • Compliance score e readiness report
Disponibile

Microsoft Defender for Endpoint

Gap analysis completo del tenant MDE/XDR: verifica policy AV, regole ASR, EDR onboarding, tamper protection e Secure Score rispetto alla baseline Jeffrey Appel.

  • AV Next-Gen Protection baseline (cloud block High+)
  • 16 regole ASR (audit → block mode)
  • EDR onboarding, tamper + network protection
  • Secure Score M365 + alert attivi
Disponibile

Conditional Access Baseline

Analizza le CA policy esistenti nel tenant e deploya la baseline completa di 24 policy (j0eyv/ConditionalAccessBaseline): Global, Admins, Internals, Guest.

  • CA000-CA006: MFA globale, blocco legacy auth, device code
  • CA100-CA105: MFA admin, phishing-resistant, CAE strict
  • CA200-CA209: Compliance Windows/macOS, risk-based block
  • CA400-CA404: Protezione completa guest
Disponibile

Assessment Security M365 + Azure

Framework di assessment e offensive simulation per Microsoft 365, Entra ID e Azure con report HTML executive-ready e mapping MITRE ATT&CK.

  • Assessment read-only su identity, M365 e risorse Azure
  • Simulazione attack chain non distruttiva (MITRE ATT&CK)
  • Severity scoring con evidenze e remediation
  • Report HTML completo per stakeholder tecnici ed executive
Disponibile

Azure Update Manager

Gestione patch centralizzata per VM Windows e Linux con Maintenance Configuration, finestre di manutenzione configurabili e policy di auto-patching.

  • Maintenance Configuration Weekly/Monthly
  • Classificazioni aggiornamento configurabili
  • Policy auto-assessment e auto-patching
  • Supporto Windows e Linux
