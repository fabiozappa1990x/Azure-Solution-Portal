Soluzioni Disponibili

Disponibile

Azure Monitor

Monitoraggio completo per VM Azure: dashboard KQL, alert CPU/Memoria/Disco, Data Collection Rules e policy di auto-enrollment.

  • Dashboard con 4 tab (Panoramica, Performance, Alert, Disponibilità)
  • Alert CPU, Memoria, Disco, Heartbeat
  • Azure Monitor Agent + DCR automatici
  • Policy auto-enrollment nuove VM
Disponibile

Azure Virtual Desktop

Desktop virtuale enterprise con Host Pool, Session Hosts, FSLogix per profili utente, Scaling Plan automatico e integrazione Entra ID.

  • Host Pool Pooled / Personal
  • Session Hosts Windows 11 multi-session
  • FSLogix su Azure Files (AADKERB)
  • Scaling Plan automatico
Disponibile

Azure Backup

Protezione dati enterprise con Recovery Services Vault, policy GFS (Daily/Weekly/Monthly/Yearly), soft delete e auto-protezione tramite Azure Policy.

  • Recovery Services Vault (GRS/LRS)
  • Policy GFS configurabili
  • Enhanced Policy (backup orario)
  • Auto-protezione VM con tag
Disponibile

Microsoft Defender for Cloud

Postura di sicurezza cloud con piani Defender attivabili (Server, Storage, Key Vault, ARM), contatti di sicurezza e auto-provisioning MDE.

  • Defender per Server P1/P2 (con MDE)
  • Defender CSPM
  • Auto-provisioning MDE + AMA
  • Microsoft Cloud Security Benchmark
Disponibile

Azure Update Manager

Gestione patch centralizzata per VM Windows e Linux con Maintenance Configuration, finestre di manutenzione configurabili e policy di auto-patching.

  • Maintenance Configuration Weekly/Monthly
  • Classificazioni aggiornamento configurabili
  • Policy auto-assessment e auto-patching
  • Supporto Windows e Linux
